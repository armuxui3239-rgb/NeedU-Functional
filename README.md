# NeedU Functional App

แอปพลิเคชันจัดการงานแบบง่าย ที่ใช้งานได้จริง

## คุณสมบัติ

- เพิ่มงานใหม่
- ทำเครื่องหมายงานที่เสร็จแล้ว
- ลบงาน
- บันทึกข้อมูลในเบราว์เซอร์ (localStorage)

## วิธีใช้งาน

1. เปิดไฟล์ `index.html` ในเบราว์เซอร์
2. พิมพ์งานที่ต้องการเพิ่มในช่อง input
3. คลิกปุ่ม "เพิ่ม" หรือกด Enter
4. คลิกที่ข้อความเพื่อทำเครื่องหมายว่าเสร็จแล้ว
5. คลิกปุ่ม "ลบ" เพื่อลบงาน
